Train: learn

Fit patterns and estimate model parameters. More varied training evidence usually gives the model more to learn from.

Validate: decide

Compare model settings without spending the final test. Repeated decisions can gradually overfit this set too.

Test: audit

Estimate performance on unseen cases only after the development choices are settled.
